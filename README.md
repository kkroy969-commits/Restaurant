# 🍽️ Global Family Restaurant Website

This is a simple and responsive restaurant website built using **HTML and CSS**.
It shows a restaurant homepage with a navigation bar, hero section, search box, and menu cards.

---

## 🚀 Features

* 🏠 Home page layout
* 📌 Navigation bar
* 🍕 Food menu cards
* 🔍 Search box (UI only)
* 🎨 Hover effects on buttons and cards
* 📱 Basic responsive layout using Flexbox

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox)
